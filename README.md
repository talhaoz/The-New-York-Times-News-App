# The New York Times News App
This app help users to easily follow up recent news in The New York Times.

## Project Structure
Developed by following MVVM design pattern and Clean Architecture rules.

- Jetpack Compose
- Hilt
- Retrofit
- Coil
- Coroutines
- DataStore
- Lifecycle
- Timber
